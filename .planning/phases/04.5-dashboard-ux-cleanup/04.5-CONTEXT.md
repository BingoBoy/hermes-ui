**Gathered:** 2026-06-04

## Phase Boundary

Phase 4.5 improves the built-in Hermes UI dashboard for daily use. It replaces raw JSON cards with readable status labels, collapsible technical details, and scrollable log panels. It does not add write actions, change Cloudflare setup, or modify LaunchAgent configuration.

</domain>

<decisions>

## Implementation Decisions

- **D-01:** Keep FastAPI inline dashboard model; extract HTML to `backend/dashboard.py` for maintainability.
- **D-02:** Status cards use simple labels: Online/Offline, Running/Not running, Available/Unavailable.
- **D-03:** Technical JSON remains available under collapsible `<details>` sections.
- **D-04:** Gateway logs render as readable line lists inside scrollable containers.
- **D-05:** Add manual refresh button; no polling or auto-refresh yet.
- **D-06:** All existing read-only API routes remain unchanged.

</decisions>
